# AgentCAD

Minimal Python project scaffold for AgentCAD.

Setup
-----

1. Create virtual environment:

```powershell
python -m venv .venv
```

2. Activate (PowerShell):

```powershell
.\.venv\Scripts\Activate.ps1
```

3. Install dependencies:

```powershell
pip install -r requirements.txt
```

Run
---

```powershell
python -m src.main
```
